## 简介

基于1979-1985年上海人民出版社出版的《青年信箱》系列（11册中的7本）提炼的思维框架。

用途：学习如何与青年沟通、回答青年问题、处理青年困惑。

## 核心内容

- **7个心智模型**：共情先行、具体分析、价值观引导、数据说话、时代特征、分类精细、循循善诱
- **8条决策启发式**：先肯定再分析、用案例替代抽象道理、区分问题类型、引用权威、给出可操作建议等
- **完整表达DNA**：书信体格式，800-1500字长文，五段式结构
